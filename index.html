Python爬虫学习之路（1）

爬取我的博客主页面的源代码

1.向发送服务器发送请求

要获取网页中的相关信息，首先应当向服务器发送一个request。
导入^request包以使用该功能： 

import request

2.使用get()方法获取网页信息

url = 'https://golden-pigeon.github.io/'
strhtml = requests.get(url)//strhtml储存了网页的信息

3.输出网页源代码

strhtml对象的text成员储存了网页的源代码
用print()函数将其输出即可 

t=strhtml.text
print(t)

4.完整程序

import requests
url = 'https://golden-pigeon.github.io/'
strhtml = requests.get(url)
t=strhtml.text
print(t)
